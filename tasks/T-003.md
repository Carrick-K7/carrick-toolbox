# T-003 - 时间转换器 + 密码生成器

> Task v1.0 | Carrick Toolbox
> 创建时间: 2025-02-06

---

## 目标

开发时间转换器和密码生成器两个实用工具，提供时间戳/时区转换和安全密码生成功能。

---

## 验收标准（必须可测试）

### Time Converter（时间转换器）
- [x] 时间戳与日期时间双向转换
- [x] 支持多种时间戳格式（秒/毫秒/微秒）
- [x] 时区转换和当前时区自动检测
- [x] 多种日期格式输出
- [x] 实时时钟显示
- [x] 历史时间计算（相对时间）

### Password Generator（密码生成器）
- [x] 随机密码生成，支持长度配置
- [x] 字符类型选择（大小写/数字/符号）
- [x] 密码强度实时评估
- [x] 排除易混淆字符选项
- [x] 批量生成多个密码
- [x] 一键复制功能

---

## 技术方案（极简）

- **Time Converter**: Intl.DateTimeFormat API处理时区，Date对象处理时间计算，自动检测时间戳精度
- **Password Generator**: Crypto.getRandomValues安全随机数，zxcvbn库评估密码强度，Clipboard API复制
- **详见**: tech_spec.md#工具实现
