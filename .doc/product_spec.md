# Product Spec - Carrick Toolbox

## Core Concept (核心概念 & Vibe)
Carrick 工具箱是一个现代化的多合一网页实用工具集合，致力于为用户提供简洁、高效、美观的日常工具体验。采用清新的青绿色主题配色，支持明暗模式切换，响应式设计适配各种设备。产品定位为"优雅的数字瑞士军刀"，让用户在一个界面中完成多种常见任务。

## User Stories (用户故事)

### 主要用户画像
- **开发者/设计师**：需要快速转换颜色格式、时间计算、编写文档
- **办公人员**：需要货币换算、时间管理、文档编辑
- **学生**：需要学习辅助工具、文档整理、时间规划

### 核心用户故事
1. 作为一个开发者，我希望能够快速转换颜色格式（HEX、RGB、HSL），以便在设计工作中使用
2. 作为一个跨国商务人士，我希望能够实时换算货币，以便进行国际业务
3. 作为一个内容创作者，我希望有一个支持实时预览的Markdown编辑器
4. 作为一个时间管理者，我希望有时钟显示和时区转换功能
5. 作为一个普通用户，我希望界面美观且支持暗色模式，保护我的眼睛

## Feature List (功能列表)

#### ⏰ 时钟工具
- [x] **模拟时钟显示**：SVG绘制，实时更新，优雅动画
- [x] **时区支持**：展示/切换时区，支持重置为本地时区
- [x] **农历显示**：正确的农历算法（1900-2100）
- [x] **节假日倒计时**：法定节假日计算
- [x] **日历功能**：农历显示、调休标记（休/班）
- [ ] **节气显示**：节气计算

- [x] **配置面板**：Switch开关样式，箭头方向优化

#### ⏰ 时间工具
- [x] **时间转换工具**：时区转换，时间戳计算，格式化显示
- [x] **字体美化**：使用Roboto Mono等宽字体

#### 🎨 颜色工具
- [x] **颜色格式转换器**：HEX/RGB/HSL/HSV/CMYK格式互转
- [x] **RGB滑动条**：实时渐变光谱背景，默认颜色背景修复
- [x] **颜色名称查询**：集成color-name-list，精确匹配30000+颜色
- [x] **屏幕取色器**：EyeDropper API支持
- [x] **UI重构**：现代化预览区，颜色名称显示

#### 💰 货币工具
- [x] **货币汇率换算**：Frankfurter API汇率数据，多货币支持
- [x] **双向换算显示**：同时显示正向和反向汇率，统一字体
- [x] **配置面板**：Switch开关样式，箭头方向优化
- [x] **自定义汇率**：支持手动修改汇率，一键恢复默认
- [x] **数据来源标注**：显示汇率数据来源信息
- [x] **快速参考优化**：货币缩写显示，标题右对齐，宽度优化
- [x] **数字字体**：系统原生等宽数字，清晰易读
- [x] **简化设计**：移除快捷金额按钮，默认金额为1

#### 📝 写作工具
- [x] **Markdown编辑器**：实时预览，工具栏，语法高亮
- [ ] **Markdown文件保存**：支持保存为 Markdown、PDF 文件

#### 🔧 开发者工具
- [ ] **正则表达式测试器**：支持正则表达式测试，支持保存正则表达式
- [ ] **JSON格式化工具**：支持JSON格式化，支持保存JSON数据

#### ⚙️ 系统功能
- [ ] **数据持久化**：保存用户配置，以及最新一篇 Markdown 文档
- [ ] **多语言支持**：中英文界面切换
- [x] **明暗主题切换**：CSS变量实现，平滑切换动画
- [x] **响应式布局**：移动端适配，触摸优化
- [x] **未实现模块提示**：开发中页面，动画效果
- [x] **侧边栏优化**：未开发模块灰色显示，收起状态完美居中

## Global Rules (全局业务规则)

1. **用户体验优先**：所有功能必须保持界面简洁、操作直观
2. **性能要求**：工具切换响应时间 < 200ms，实时计算延迟 < 50ms
3. **数据安全**：用户数据仅在本地存储，不上传到服务器
4. **兼容性**：支持现代浏览器（Chrome 90+、Firefox 88+、Safari 14+）
5. **移动优先**：所有功能必须在移动设备上完美可用
6. **无障碍访问**：支持键盘导航和屏幕阅读器
7. **主题一致性**：所有工具必须统一使用设计系统和主题变量
