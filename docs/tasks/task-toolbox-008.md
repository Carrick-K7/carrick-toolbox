# task-toolbox-008: Toolbox 时钟增强

## 需求
1. 时钟支持进一步放大、占满容器
2. 时钟支持进入沉浸模式（不熄屏）

## 技术栈
- Vue 3 + Vite + pnpm + Tailwind v3 + carrick-ui

## 实现
1. 放大功能：
   - 添加全屏按钮（左上角）
   - 支持双击时钟进入/退出全屏
   - 全屏时自动增大时钟尺寸
   - 支持全屏API和CSS降级方案

2. 沉浸模式：
   - 添加沉浸模式按钮（左上角）
   - 使用 Screen Wake Lock API 防止熄屏
   - 页面重新可见时自动重新获取锁
   - 退出工具时自动释放锁
